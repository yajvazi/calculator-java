# Kalkulator i Thjeshte ne Java / Simple Calculator in Java

## ALB

**Pershkrimi:** Studentet krijojne nje kalkulator qe kryen operacione bazike matematikore si mbledhje, zbritje, shumezim dhe pjestim.

**Qellimi:** Te mesojne perdorimin e metodave, kushteve dhe hyrjes se te dhenave nga perdoruesi.

## ENG

**Description:** Students create a calculator that performs basic mathematical operations such as addition, subtraction, multiplication, and division.

**Objective:** To learn the use of methods, conditions, and user input.

## How to Run

```bash
javac SimpleCalculator.java
java SimpleCalculator
```

## Example

```text
Choose operation: 1
First number: 10
Second number: 5
Result: 15.0
```
